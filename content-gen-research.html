Google и AI-контент в 2026

Главный вывод

Google НЕ наказывает AI-контент как таковой. Наказывает: mass-produced без экспертизы, без E-E-A-T, без человеческого review. Bankrate открыто использует AI + human expert review и ранжируется.

Helpful Content System

  • С марта 2024 интегрирована в core ranking (не отдельный апдейт)
  • Information Gain Score — оценивает уникальную ценность страницы vs конкуренты
  • Оценка на уровне всего сайта — плохой контент тянет вниз всё
  • С июня 2025 — manual actions (полное удаление) за scaled content abuse

Ahrefs AI Detection

  • Accuracy: 80-85%, false positives: 0-8%
  • Bulk detection через Site Audit (Pro plan, 1K pages/crawl)
  • Обходится существенной человеческой редактурой
  • Отдельного API endpoint для AI detection нет

Архитектуры генерации

A: Claude API + Codex Review

Claude генерирует статью → Codex/второй Claude ревьюит → итерация до quality threshold.

Стоимость/1000 статей$200-1,200
Время1-3 дня
Качество7-8/10
AI detection рискВысокий
Масштабируемость9/10

B: Human-in-the-Loop (HITL)

AI draft → эксперт review (10-15 мин) → publish с byline. Industry standard NerdWallet/Bankrate.

Стоимость/1000 статей$5,000-10,000
Время2-4 недели
Качество8-9/10
AI detection рискНизкий
Масштабируемость4/10 (bottleneck = люди)

C: Self-hosted LLM (Ollama/vLLM)

Llama 3.3 70B / Qwen 2.5 72B на арендованном GPU. Fine-tune на стиле эксперта (LoRA).

Стоимость/1000 статей$50-200 (GPU time)
Время1-2 дня
Качество5-7/10 (без fine-tune) / 6.5-7.5 (с fine-tune)

Рекомендация: Гибрид D+B (n8n + Light HITL)

Оптимальная архитектура для финансового контента:

  1. n8n на Contabo — полный pipeline: keyword → outline → research → draft → compliance check → SEO
  2. Claude Sonnet API — генерация ($0.08/статья), лучший баланс цена/качество
  3. Auto compliance check — второй вызов Claude с чеклистом
  4. Light human review — эксперт 10-15 мин на финальную проверку
  5. WordPress REST API — publish с Schema.org (Article + Review + FAQPage)

Минимальный объём для SEO

  • Менее 2,500 слов — низкие шансы конкурировать по коммерческим keywords (наблюдение, не правило)
  • 4,000-5,000 слов — типичный объём у конкурентов (ForexBrokers, BrokerChooser)
  • 6,000-10,000 слов — наблюдается у top-ranked обзоров (BrokerChooser). NerdWallet ранжируется и с меньшим объёмом

Schema.org (обязательный минимум)

  • Review + Product + Article в @graph
  • positiveNotes / negativeNotes как ItemList
  • FAQPage для FAQ секции (Google ограничил rich results для FAQ в 2023, но schema всё ещё используется топ-сайтами)
  • BreadcrumbList
  • Person с sameAs на LinkedIn для авторов

Что работает в финансах (кейсы)

Bankrate — УСПЕХ

Открыто раскрыли AI-assisted content. AI draft + human expert edit + "AI-assisted" маркировка. Продолжают ранжироваться по конкурентным keywords.

HubSpot — ПРОВАЛ

Organic traffic упал на 70-80% (13.5M → 7M visits). Generic high-volume content strategy без expertise сигналов.

Формула успеха

50-100 качественных AI+human статей = +30-80% трафика.
1000+ unedited AI статей = -40-90% трафика.

Ключ: AI как инструмент черновика + человеческая экспертиза + E-E-A-T сигналы.
